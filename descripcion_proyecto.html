Descripción Completa del Proyecto

1. Descripción General

Plantilla web profesional para locales comerciales. El proyecto base es para el restaurante 'La Tragadera'.

Stack Tecnológico: Next.js, React, TypeScript, Tailwind CSS.

Librerías de UI

  • Shadcn/UI: Colección de componentes de interfaz de usuario construidos con React y Tailwind CSS, diseñados para ser personalizables y accesibles. Utilizado para elevar la calidad visual y funcional de los componentes clave del proyecto.

2. Estructura de Archivos

La estructura del proyecto se configuró manualmente para seguir las mejores prácticas. La estructura principal es:

  • /src: Contiene todo el código fuente de la aplicación.
    • /app: Contiene los archivos de enrutamiento, páginas y layouts de la aplicación (App Router). Cada subcarpeta representa una ruta.
      • / (raíz): Contiene el layout global (layout.tsx), la página de inicio (page.tsx) y los estilos globales (globals.css). El layout.tsx ahora incluye metadatos SEO, integración con Google Analytics y el componente FloatingWhatsAppButton.
      • /menu: Página para el menú del restaurante.
      • /contacto: Página para la información de contacto.
    • /components: Carpeta para todos los componentes de React reutilizables.
      • /layout: Componentes relacionados con la estructura de la página (Header, Footer, Sidebar, etc.).
      • /ui: Pequeños componentes de interfaz de usuario (Botones, Inputs, Cards, etc.).
    • /data: Almacena datos estáticos de la aplicación.
      • menu.ts: Define la estructura de datos para los productos del menú.
      • data.ts: Contiene los arrays menuCategories (categorías del menú con imágenes) y promociones (combos con título, descripción, precio e imagen).
  • /public: Para almacenar archivos estáticos como imágenes, fuentes, etc.
  • /node_modules: Contiene todas las dependencias del proyecto instaladas vía npm.
  • package.json: Define los scripts del proyecto (dev, build, etc.) y las dependencias.
  • next.config.mjs: Archivo de configuración para Next.js.
  • tailwind.config.ts: Archivo de configuración para Tailwind CSS.
  • tsconfig.json: Archivo de configuración para TypeScript.
  • postcss.config.mjs: Archivo de configuración para PostCSS, usado por Tailwind.
  • .gitignore: Especifica los archivos y carpetas que Git debe ignorar.

3. Componentes de la Interfaz (UI)

Header.tsx

Ubicación: src/components/layout/Header.tsx

Propósito: Es la barra de navegación principal, fija en la parte superior y totalmente responsiva. Reconstruido con el componente NavigationMenu de Shadcn/UI para los enlaces de navegación y el componente Button de Shadcn/UI para el llamado a la acción de WhatsApp.

Funcionalidades:

  • Es un Componente de Cliente ('use client') que usa estado para el menú móvil.
  • Muestra los enlaces de navegación principales en pantallas de escritorio.
  • En pantallas pequeñas, muestra un botón de "hamburguesa" que despliega un menú vertical.
  • Utiliza iconos de la librería react-icons.

HeroSection.tsx

Ubicación: src/app/components/HeroSection.tsx

Propósito: Componente principal de la página de inicio que muestra una imagen de fondo impactante y un llamado a la acción.

Funcionalidades:

  • Utiliza next/image para una imagen de fondo optimizada que ocupa el 60% de la altura visible (h-[60vh]).
  • Presenta un eslogan principal con una sombra de texto (text-shadow-heavy) para mayor legibilidad e impacto.
  • Incluye un subtítulo descriptivo para mayor contexto y jerarquía tipográfica.
  • El fondo cuenta con un degradado sutil para añadir profundidad visual.
  • Incluye un botón de llamado a la acción para pedir por WhatsApp.

Footer.tsx

Ubicación: src/components/layout/Footer.tsx

Propósito: Es el pie de página principal del sitio.

Funcionalidades:

  • Muestra el año de copyright dinámicamente.
  • Contiene placeholders para enlaces a redes sociales.
  • Se mantiene al final de la página gracias a la estructura Flexbox del layout principal.

Button.tsx

Ubicación: src/components/ui/Button.tsx

Propósito: Componente de botón reutilizable para toda la aplicación, ahora basado en el componente Button de Shadcn/UI, ofreciendo mayor accesibilidad y opciones de personalización.

Props:

  • children: Contenido del botón.
  • variant ('primary' | 'secondary'): Define el estilo visual.
  • Acepta todos los atributos estándar de un botón HTML.

MenuItemCard.tsx

Ubicación: src/components/ui/MenuItemCard.tsx

Propósito: Componente de tarjeta para mostrar un producto del menú. Su funcionalidad principal ha sido reemplazada por el componente Card de Shadcn/UI en la página de menú para una presentación más profesional.

Props:

  • item (MenuItem): Objeto con la información del producto.

Funcionalidades:

  • Muestra la imagen del producto utilizando next/image, con optimización de sizes y quality.
  • La descripción del producto ha sido mejorada con un estilo más legible (color y tamaño de fuente).
  • Incluye un botón de "Añadir" (actualmente sin funcionalidad).
  • Tiene un efecto de zoom en la imagen al pasar el cursor.

MenuSection.tsx

Ubicación: src/app/components/MenuSection.tsx

Propósito: Componente encargado de renderizar la lista de productos del menú utilizando los componentes Card de Shadcn/UI para una presentación profesional y estructurada.

Funcionalidades:

  • Mapea sobre los elementos del menú y renderiza cada uno dentro de un componente Card.
  • Utiliza CardHeader para el nombre y descripción del producto, CardContent para la imagen y CardFooter para el precio y un botón de acción.

ContactForm.tsx

Ubicación: src/components/ui/ContactForm.tsx

Propósito: Formulario de contacto interactivo y reutilizable.

Funcionalidades:

  • Componente de Cliente ('use client') que maneja el estado de los campos.
  • Realiza una acción de envío (actualmente, a la consola) y muestra un mensaje de éxito.

AnimationWrapper.tsx

Ubicación: src/components/ui/AnimationWrapper.tsx

Propósito: Anima a sus hijos para que aparezcan con un efecto de "fade-in" al hacer scroll.

BannerCarousel.tsx

Ubicación: src/components/layout/BannerCarousel.tsx

Propósito: Muestra una secuencia de imágenes de banner de forma interactiva y animada.

Funcionalidades:

  • Es un Componente de Cliente ('use client') que gestiona el banner actual.
  • Cambia automáticamente de banner cada 5 segundos.
  • Utiliza framer-motion para transiciones suaves entre banners.
  • Incluye puntos de navegación para control manual.

PromotionsSection.tsx

Ubicación: src/app/components/PromotionsSection.tsx

Propósito: Muestra una cuadrícula de promociones del restaurante utilizando componentes Card de Shadcn/UI.

Funcionalidades:

  • Obtiene los datos de las promociones desde src/lib/data.ts.
  • Renderiza cada promoción como una tarjeta con imagen, título y descripción, con optimización de sizes y quality para las imágenes.
  • La descripción de la promoción ha sido mejorada con un estilo más legible (color y tamaño de fuente).
  • Se añadió min-h-0 al contenedor de la imagen para asegurar su correcta visualización.
  • Incluye un ID #promos para la navegación por scroll.

CtaSection.tsx

Ubicación: src/app/components/CtaSection.tsx

Propósito: Sección de llamado a la acción para incentivar a los usuarios a realizar un pedido.

Funcionalidades:

  • Presenta un eslogan impactante con la fuente Anton.
  • Incluye un botón grande de "Pide por WhatsApp" que enlaza directamente al chat.
  • Tiene un fondo de color de acento (amarillo) para destacar.

PromotionalCarousel.tsx

Ubicación: src/app/components/PromotionalCarousel.tsx

Propósito: Muestra un carrusel dinámico y automático con banners de promociones.

Funcionalidades:

  • Utiliza el componente Carousel de Shadcn/UI.
  • Incluye el plugin Autoplay para rotar las imágenes cada 5 segundos.
  • Muestra las imágenes banner-largo-horizontal-promo.jpg y banner-promocion.jpg, con optimización de sizes y quality.
  • Se añadió min-h-0 al contenedor de la imagen para asegurar su correcta visualización.
  • Es responsivo y se adapta al ancho del contenedor.

SmallFlyer.tsx

Ubicación: src/app/components/SmallFlyer.tsx

Propósito: Muestra una imagen estática (flyer) para añadir interés visual y comunicar promociones o eventos específicos.

Funcionalidades:

  • Utiliza el componente Image de Next.js para mostrar /images/banner-rotativo.jpg, con optimización de quality.
  • Se añadió min-h-0 al contenedor de la imagen para asegurar su correcta visualización.
  • Está centrado y tiene un margen vertical para separarlo de otras secciones.

FloatingWhatsAppButton.tsx

Ubicación: src/components/ui/FloatingWhatsAppButton.tsx

Propósito: Botón flotante e interactivo para contactar vía WhatsApp.

Funcionalidades:

  • Es un Componente de Cliente ('use client') para manejar la visibilidad y el mensaje.
  • Aparece después de un retardo y muestra un mensaje emergente temporal.
  • Se posiciona de forma fija en la esquina inferior derecha de la pantalla.
  • Utiliza un icono de WhatsApp de react-icons.
  • Enlaza directamente al chat de WhatsApp del negocio.
