Descripción Completa del Proyecto

1. Descripción General

Plantilla web profesional para locales comerciales. El proyecto base es para el restaurante 'La Tragadera'.

Stack Tecnológico: Next.js, React, TypeScript, Tailwind CSS.

2. Estructura de Archivos

La estructura del proyecto se configuró manually para seguir las mejores prácticas. La estructura principal es:

  • /src: Contiene todo el código fuente de la aplicación.
    • /app: Contiene los archivos de enrutamiento, páginas y layouts de la aplicación (App Router). Cada subcarpeta representa una ruta.
      • / (raíz): Contiene el layout global (layout.tsx), la página de inicio (page.tsx) y los estilos globales (globals.css).
      • /menu: Página para el menú del restaurante.
      • /contacto: Página para la información de contacto.
    • /components: Carpeta para todos los componentes de React reutilizables.
      • /layout: Componentes relacionados con la estructura de la página (Header, Footer, Sidebar, etc.).
      • /ui: Pequeños componentes de interfaz de usuario (Botones, Inputs, Cards, etc.).
    • /data: Almacena datos estáticos de la aplicación, como la información del menú (menu.ts).
  • /public: Para almacenar archivos estáticos como imágenes, fuentes, etc.
  • /node_modules: Contiene todas las dependencias del proyecto instaladas vía npm.
  • package.json: Define los scripts del proyecto (dev, build, etc.) y las dependencias.
  • next.config.mjs: Archivo de configuración para Next.js.
  • tailwind.config.ts: Archivo de configuración para Tailwind CSS.
  • tsconfig.json: Archivo de configuración para TypeScript.
  • postcss.config.mjs: Archivo de configuración para PostCSS, usado por Tailwind.
  • .gitignore: Especifica los archivos y carpetas que Git debe ignorar.

3. Componentes de la Interfaz (UI)

Header.tsx

Ubicación: src/components/layout/Header.tsx

Propósito: Es la barra de navegación principal, fija en la parte superior y totalmente responsiva.

Funcionalidades:

  • Es un Componente de Cliente ('use client') que usa estado para el menú móvil.
  • Muestra los enlaces de navegación principales en pantallas de escritorio.
  • En pantallas pequeñas, muestra un botón de "hamburguesa" que despliega un menú vertical.
  • Utiliza iconos de la librería react-icons.

Footer.tsx

Ubicación: src/components/layout/Footer.tsx

Propósito: Es el pie de página principal del sitio.

Funcionalidades:

  • Muestra el año de copyright dinámicamente.
  • Contiene placeholders para enlaces a redes sociales.
  • Se mantiene al final de la página gracias a la estructura Flexbox del layout principal.

Button.tsx

Ubicación: src/components/ui/Button.tsx

Propósito: Componente de botón reutilizable para toda la aplicación.

Props:

  • children: Contenido del botón.
  • variant ('primary' | 'secondary'): Define el estilo visual.
  • Acepta todos los atributos estándar de un botón HTML.

MenuItemCard.tsx

Ubicación: src/components/ui/MenuItemCard.tsx

Propósito: Tarjeta para mostrar un producto del menú de forma atractiva.

Props:

  • item (MenuItem): Objeto con la información del producto.

ContactForm.tsx

Ubicación: src/components/ui/ContactForm.tsx

Propósito: Formulario de contacto interactivo y reutilizable.

Funcionalidades:

  • Componente de Cliente ('use client') que maneja el estado de los campos.
  • Realiza una acción de envío (actualmente, a la consola) y muestra un mensaje de éxito.

AnimationWrapper.tsx

Ubicación: src/components/ui/AnimationWrapper.tsx

Propósito: Un componente de cliente reutilizable que anima a sus hijos (children) para que aparezcan con un efecto de "fade-in" y deslizamiento hacia arriba cuando entran en el viewport.

Props:

  • children: Los elementos a animar.
  • delay (opcional): Un retraso en segundos para iniciar la animación.
  • className (opcional): Clases CSS adicionales para el div contenedor.

Funcionalidades:

  • Utiliza el hook useInView de Framer Motion para detectar la visibilidad.
  • La animación se ejecuta solo una vez por defecto.

PlaceholderImage.tsx

Ubicación: src/components/ui/PlaceholderImage.tsx

Propósito: Muestra un placeholder visualmente agradable para imágenes.
